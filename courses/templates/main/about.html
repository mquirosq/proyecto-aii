Acerca de Curio

Curio es una plataforma web que centraliza cursos online de distintas plataformas ( Coursera, edX y OpenLearn) para facilitar su búsqueda, comparación y recomendación personalizada.

Objetivo del proyecto

El objetivo de Curio es ayudar a los usuarios a descubrir cursos relevantes de forma rápida e inteligente, evitando tener que buscar entre múltiples plataformas de cursos online.

Curio ha sido desarrollado como parte de la asignatura de AII, aplicando técnicas reales de recuperación de información y recomendación.

Búsqueda y exploración

Curio permite buscar cursos utilizando un motor de búsqueda basado en Whoosh, teniendo en cuenta el título, la descripción y las palabras clave de cada curso. Además, ofrece filtros avanzados por:

  • Plataforma
  • Categoría
  • Nivel
  • Duración
  • Puntuación
  • Instructor

Recomendaciones inteligentes

La plataforma incorpora un sistema de recomendación híbrido que combina:

  • Recomendación basada en contenido, utilizando características del curso (categoría, nivel, duración, palabras clave, etc.) y el historial del usuario.
  • Recomendación colaborativa, basada en la similitud entre usuarios y sus interacciones con los cursos.

Las recomendaciones se adaptan al comportamiento del usuario mediante likes, dislikes y visualizaciones (rating implícito), dando más peso a las interacciones recientes.

Tecnologías utilizadas

  • Django para el backend y la gestión de datos.
  • SQLite como base de datos.
  • Whoosh como motor de búsqueda.
  • Python para la lógica de recomendación y scraping.
  • Bootstrap para la interfaz de usuario.
