# Scraping, recuperación y recomendación de cursos online

Proyecto para la asignatura AII.

## Descripción

Este proyecto tiene como objetivo desarrollar una página web con Django que permita a los usuarios buscar, recuperar y recibir recomendaciones de cursos online de diversas plataformas educativas. La aplicación web integrará técnicas de scraping con BeautifulSoup para obtener datos actualizados sobre los cursos disponibles, así como recuperación de los datos con Whoosh y algoritmos de recomendación para sugerir cursos basados en las preferencias del usuario.
